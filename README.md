# KGFL - Кыргызская Футбольная Лига

Веб-приложение для управления футбольной лигой Кыргызстана.

## Возможности

### Для пользователей:
- Просмотр турнирной таблицы с разницей голов
- Информация о клубах с логотипами
- Детальные страницы клубов с составом, матчами и руководством
- Карточки игроков с боковой панелью информации
- Просмотр матчей и результатов

### Для администраторов:
- Управление клубами с загрузкой логотипов
- Управление игроками и тренерами
- Контроль матчей в реальном времени
- Автоматическое обновление статистики и таблицы

## Технологии

### Backend:
- Django 5.0
- Django REST Framework
- PostgreSQL
- Celery для фоновых задач

### Frontend:
- Next.js 14
- TypeScript
- Tailwind CSS
- Framer Motion для анимаций
- Lucide React для иконок

## Установка и запуск

### 1. Клонирование репозитория
```bash
git clone <repository-url>
cd KGFL
```

### 2. Настройка Backend

```bash
cd back

# Создание виртуального окружения
python -m venv venv

# Активация виртуального окружения
# Windows:
venv\Scripts\activate
# Linux/Mac:
source venv/bin/activate

# Установка зависимостей
pip install -r requirements.txt

# Настройка базы данных
python manage.py migrate

# Создание суперпользователя
python manage.py createsuperuser

# Запуск сервера
python manage.py runserver
```

### 3. Настройка Frontend

```bash
cd front

# Установка зависимостей
npm install

# Создание файла .env.local
echo "NEXT_PUBLIC_API_URL=http://localhost:8000" > .env.local

# Запуск сервера разработки
npm run dev
```

## API Endpoints

### Клубы
- `GET /api/clubs/` - Список всех клубов
- `GET /api/clubs/{id}/` - Информация о клубе
- `GET /api/clubs/{id}/players/` - Игроки клуба
- `GET /api/clubs/{id}/matches/` - Матчи клуба
- `GET /api/clubs/table/` - Турнирная таблица

### Матчи
- `GET /api/matches/` - Список всех матчей
- `PATCH /api/matches/{id}/` - Обновление матча

### Тренеры
- `GET /api/coaches/` - Список тренеров
- `GET /api/coaches/?club_id={id}` - Тренеры клуба

## Особенности реализации

### Автоматическое обновление статистики
- При изменении статуса матча на "finished" автоматически обновляется статистика клубов
- Пересчитываются очки, голы, позиции в таблице
- Используются Django сигналы для автоматизации

### Турнирная таблица
- Сортировка по очкам, разнице голов, забитым голам
- Автоматическое обновление позиций
- Отображение разницы голов с цветовой индикацией

### Админ-панель
- Управление матчами в реальном времени
- Загрузка логотипов клубов
- Контроль результатов и статусов матчей
